# 🚀 Hirely-Job-Platform

**Hirely** is a vibrant, modern, and feature-rich job board platform built with Django. It bridges the gap between hirers and job seekers with a streamlined workflow, automated contracts, and a secure payment/deposit system.

## ✨ Key Features

- **🌈 Vibrant UI**: A modern, high-chroma design system with glassmorphism and smooth transitions.
- **🤝 Automated Contracts**: Legally-binding contract initiation and acceptance workflow for both parties.
- **🛡️ Secure Deposits**: Automated advance payment system (20% budget) to ensure commitment.
- **💬 Real-time Messaging**: Integrated chat system for seamless communication between hirers and seekers.
- **📝 Daily Updates**: A structured feedback loop with daily progress reports and file attachments.
- **🔔 Smart Notifications**: Real-time alerts for job applications, contract status, and messages.
- **⭐ Reviews & Ratings**: Mutual rating system to build trust within the community.
- **🚩 Reporting System**: Built-in safety features to report issues or suspicious activity.

---

## 🛠️ Tech Stack

- **Backend**: Python 3.x, [Django 6.0](https://www.djangoproject.com/)
- **Database**: SQLite (default)
- **Frontend**: Vanilla CSS (Custom Vibrant Design System), JavaScript
- **Static Assets**: Django Static Files

## 🚀 Getting Started

### Prerequisites

- Python 3.10+
- pip (Python package manager)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/Hirely-Job-Platform.git
   cd Hirely-Job-Platform
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run migrations**:
   ```bash
   python manage.py migrate
   ```

5. **Start the development server**:
   ```bash
   python manage.py run server
   ```

6. **Access the application**:
   Open `http://127.0.0.1:8000` in your browser.
